# 🗄️ CodePixel Arena – Database

This repository contains the **database implementation** of the CodePixel Arena project.

It includes:
- Database schema (tables, relationships)
- SQL scripts (DDL, DML, DQL)
- Views, indexes, and constraints
- Triggers and stored procedures

The database supports core features such as users, challenges, submissions, and pixel ownership.
